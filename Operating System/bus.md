[[OS program]] [[system bus]]

A computer bus acts as the internal data highway, routing communication between system components. The CPU and main memory are tightly coupled via the system bus, which bridges these core components directly to the motherboard. Because CPUs process instructions much faster than main memory can serve them, the system bus is designed to operate at frequencies significantly higher than the memory modules to prevent data starvation and bottlenecking.
#### system bus
- connect internal component of a computer to the mother board.
