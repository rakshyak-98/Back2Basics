## Installation

[doc](https://pkl-lang.org/main/current/pkl-cli/index.html)


```bash
pkl eval app.pkl;
pkl eval --format json app.pkl;
pkl eval --format yaml app.pkl;
```

- multiple environment 

```bash
pkl eval -p env=production app.pkl;
```
