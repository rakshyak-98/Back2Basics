Open Broadcaster Software -> is a free, open-source software for capturing, encoding and streaming live video. 

> [!INFO]
> OBS streams to your [[ingest server]] via RTMP

**Primary functions**
- capture video/audio from sources (webcam, screen, game RTMP input)
- Compose scenes (mix multiple sources, and overlays, text)
- Encode in real-time
- Stream to RTMP server or record locally
