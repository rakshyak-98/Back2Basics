```bash
set clipboard=unnamedplus
set expandtab
set shiftwidth=4 # indentation commands >>, << shift by 4 spaces.
set tabstop=4 # pressing <Tab> inserts 2 spaces because of expandtab.
set expandtab # tabs are converted to spaces.

```
